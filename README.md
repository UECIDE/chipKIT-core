# chipkit-core
Downloadable chipKIT core for use with Arduino 1.6x IDE and UECIDE

## Build Instructions

```
ant setup
ant build
ant macosx-build
ant windows-build
ant linux32-build
ant linux64-build
ant raspberrypi-build
ant clean
ant updatepic32prog
```

## Deployment instructions
```
Copy from dist OS Platform and move to Arduino/hardware folder
```
